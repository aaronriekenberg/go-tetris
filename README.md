# go-tetris

TUI tetris in go using [tcell](https://github.com/gdamore/tcell).

## Keys:

- Escape, Q: quit
- R: restart
- Left Arrow: move piece left
- Right Arrow: move piece right
- Up Arrow: rotate piece
- Down Arrow: move piece down
- Space Bar: drop piece
